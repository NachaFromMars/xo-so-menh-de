# Hệ Thống Đài — 60+ Đài, 10 Vùng Cược

## 10 Vùng Cược (Region / Play Style)

| Region | Play Style | Mô tả |
|--------|-----------|-------|
| MN (Miền Nam) | `mn` | Miền Nam default — 21 đài |
| MN | `mn_mtay` | MN Miền Tây |
| MN | `mn_buonme` | MN Buôn Mê |
| MT (Miền Trung) | `mt` | Miền Trung default — 11 đài |
| MT | `mt_mtay` | MT Miền Tây |
| MT | `mt_buonme` | MT Buôn Mê |
| MB (Miền Bắc) | `mb` | Miền Bắc default |
| MB | `mb_hanoi` | MB Hà Nội (riêng) |
| MB | `mb_ngan` | MB Ngắn |
| MB | `mb_buonme` | MB Buôn Mê |

## Danh Sách Đài

### MIỀN NAM (21 đài)
TP HCM, Đồng Tháp, Cà Mau, Bến Tre, Vũng Tàu, Bạc Liêu, Đồng Nai, Cần Thơ, Sóc Trăng, Bình Thuận, Tây Ninh, An Giang, Bình Dương, Trà Vinh, Vĩnh Long, Tiền Giang, Kiên Giang, Đà Lạt (Lâm Đồng), Long An, Bình Phước, Hậu Giang

### MIỀN TRUNG (11+ đài)
Huế (Thừa Thiên Huế), Đà Nẵng, Phú Yên, Khánh Hòa, Quảng Bình, Quảng Trị, Quảng Nam, Quảng Ngãi, Bình Định, Gia Lai, Kon Tum, Đắk Lắk, Đắk Nông, Ninh Thuận

### MIỀN BẮC
Hà Nội (đài chính) — xổ hàng ngày. Các đài phụ theo biến thể MB Ngắn, MB Miền Tây, MB Buôn Mê.
